Правила игры:

	Игра происходит на квадратном поле, состоящем из маленьких квадратов. Игроки-боты (исполняемые файлы) вызываются по очереди программой-оболочкой (данной программой). 
	При вызове они получают данные о поле и передают назад данные о своем ходе (см. Сообщение игрового алгоритма и приложения). Ход игры - поставить фигуру, которую вы указываете в 
	настройках фигуры (см. Настройки фигуры). Тот, кто не может сделать ход, делает невозможный ход (выходит за границу поля или задевает уже занятую клетку,
	или же не проходит ограничение на время выполнения (см. Настройки:Ограничение на время выполнения)) - тот проигрывает. Задача каждого из игроков-ботов - победить - сделать так,
	чтобы другой игровой алгоритм проиграл. 

Запуск:
Запустить игру можно 2-мя способами: 
	1. Обычная игра - контроль каждого хода. Нажмите "Начать". По кнопке "Сделать ход" вы запускаете игрока, который должен ходить следующим.
	2. Быстрая игра - сессия с заданным кол-вом игр. Нажмите "Быстрая игра" и выберите бегунком кол-во игр в сессии.

Статистика:
Нажав кнопку статистика, вы увидете:
	Кол-во сыгранных партий - сколько партий сыгранно с последнего очищения статистики. 
	Далее двумя строками идут проценты побед игроков. В начале написаны их названия.
	Для того, чтобы очистить статистику:
		1. Зайдите в настройки
		2. Нажмите на кнопку "Очистить"
		3. Если очистка статистики прошла успешно, вы увидете надпись "Stats cleaned"

Настройки:

	Ограничение на время выполнения:
		Текстовое поле "Limit time for step (ms)" - максимальное время ожидания хода игрока в миллисекундах. Если игрок не делает ход в течение этого времени, то он проигрывает.

	Изменение размера поля:
		Бегунок "Rectangle size" - размер стороны квадрата-поля. Возможно задать от 5 до 40.
		Из-за того, что проводить много боев для того, чтобы иметь большую выборку - очень долго, поэтому предлагается менять размеры поля. Оптимальные размеры - 20х20 или 35х35. 
		Но для улучшения выборки - уменьшения вероятностных поражений имеет смысл менять размер поля между баталиями.

	Подгрузка .ехе-файлов игроков:
		Далее идут два текстовых поля - ссылки на .ехе-файлы игроков. Для того, чтобы начать писать нажмите на текстовое поле, для выхода из режима редактирования - нажмите esc 
		или кликните мышкой вне текстового поля. Для ввода - нажмите enter. Также для того, чтобы сохранить вашу ссылку, вы можете нажать "Apply" или "Enter" (на клавиатуре).
		
		!ВАЖНО! Ссылки должны быть полным путем файла и содержать только английские символы а так же простые знаки (!, @, #, $, %, ^, &, *, (, ), :, ", ;, \, /, ., _, +, =) !ВАЖНО!

!ВАЖНО! Сообщение игрового алгоритма и приложения: !ВАЖНО!

	На выход игровой алгоритм получает файл map.txt, который он может найти по тому пути, по которому его вызвали. Например: file = open(map.txt, "r") - (python). Файл map.txt 
	систематизарован таким образом: первая строчка - параметры фигуры (9 единиц и нулей через пробел. По очереди по "строчкам" записанные. 0 - клетка).
	Вторая строчка - размер квадрата поля (кол-во клеток на одной стороне). Далее на следующей строчке через пробел даны значения клеток 
	(1 - занято, 0 - свободно). Клетки идут поочередно по строкам (все подряд, но сначала записанна первая "строка" поля, затем вторая и т.д.). 
	
	Данное приложение автоматически запускает игрового алгоритма и автоматически создает необходимые файлы. После того, как игровой алгоритм получил и обработал данные из файла map.txt, 
	он должен вернуть программе-оболочке данные о своем ходе (позицию х и у центра фигуры, которую он ставит на поле, а также поворот (0 - 0, 1 - 90, 2 - 180, 3 - 270) по часовой 
	стрелке). Для этого запишите в файл result.txt (который лежит там же, где и map.txt) 2 числа (х, затем у через пробел, а затем - поворот числом). 
	
Настройка фигуры:
	Нажав кнопку "Edit" вы можете поменять фигуру, которую игровые алгоритмы должны ставить на поле. Там вы увидите 9 квадратов-кнопок, которые символизируют 9 клеток фигуры. 
	Если этот квадрат белый, то фигуры в этом месте нет, если квадрат синий, то фигура в этом месте есть. Так из занятых и не занятых клеток рисуется фигура.
	!ВАЖНО! Чтобы сохранить вашу фигуру нажмите "Save"! Если вы просто выйдите в меню, ваша фигура не сохранится.
